## Download Memgraph

* [Source](https://memgraph.com/download)
* Using Docker image:

  ```bash
  docker run -it -p 7687:7687 -p 7444:7444 -p 3000:3000 memgraph/memgraph-platform
  ```

* Connect to Memgraph Lab: `localhost:3000`
* Memgraph DB backend: `memgraph://127.0.0.1:7687`
